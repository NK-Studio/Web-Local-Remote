# Web Local Remote

Web Local Remote는 유니티에서 빌드한 WebGL 프로젝트를 로컬 서버를 통해 구동하고, 같은 와이파이(Wi-Fi) 내의 모바일 기기나 다른 PC에서 간편하게 접속하여 테스트할 수 있도록 돕는 에디터 플러그인입니다.

## 🚀 주요 기능
- **원클릭 로컬 서버**: 복잡한 환경 설정 없이 버튼 하나로 WebGL 로컬 서버를 구동할 수 있습니다.
- **자동 Node.js 연결**: 외부 Node.js 설치가 필요 없습니다. 유니티에 내장된 WebGL Support 도구를 자동으로 찾아서 활용합니다.
- **모바일 QR 코드 테스트 지원**: 서버 구동 시 제공되는 QR 코드를 스마트폰으로 스캔하기만 하면 즉시 모바일 디바이스에서 동작을 확인할 수 있습니다.
- **편리한 UI 요소**: 깔끔하게 구조화된 다크 모드 적응형 UI 툴킷과 그룹 박스 스타일을 제공합니다.

## 🛠 사용 방법
1. 유니티 엔진 상단 탭에서 **`Tools > Web Local Remote`** 메뉴를 엽니다.
2. WebGL 모듈이 설치되어 있는지 확인합니다. 
   *(경고 뷰가 나타난다면 Unity Hub에서 현재 버전에 대한 명시적인 WebGL Build Support 모듈을 시스템에 인스톨해주세요.)*
3. **`Browse Folder`** 버튼을 누르시고 WebGL 프로젝트가 빌드된 위치(폴더)를 지정합니다. 지정된 경로는 에디터가 자동으로 기억합니다!
4. 하단의 **`▶ Run`** (토글 버튼)를 클릭하여 서버를 구동시킵니다.
5. 하단 영역에 열린 **QR 코드**를 모바일 기기의 카메라나 스캐너 앱을 이용해 스캔하거나, 표시된 `http://...` 링크를 복사하여 브라우저에서 직접 테스트합니다.
6. 테스트를 마무리지으시려면 토글된 버튼인 **`■ Stop`** 버튼을 다시 한 번 눌러 서버를 끕니다.

## ⚠️ 유의 사항
- 서버 구동 후 표시되는 IP 주소는 **동일한 로컬 네트워크(동일 LAN/Wi-Fi)** 환경에서만 유효합니다.
- 플러그인 뷰 창을 닫아도 구동시킨 서버의 세션과 포트는 유지됩니다. 원치 않게 포트 충돌이 생길 수 있으므로 작업을 마쳤을 때는 반드시 UI상에서 `■ Stop` 버튼을 눌러주세요.
